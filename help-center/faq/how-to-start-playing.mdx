## Шаги для старта

1. Перейдите в Telegram и найдите <b>@CometFall_bot</b>.
2. Нажмите «Start».
3. Следуйте подсказкам бота для создания персонажа.

## Основные команды
- <code>/profile</code> — посмотреть свой профиль.
- <code>/battle</code> — начать бой с врагом.
- <code>/shop</code> — открыть магазин для покупки оружия и предметов.

## Важно
<Callout kind="info">Для игры нужен только Telegram — не требуется установка приложений.</Callout>
